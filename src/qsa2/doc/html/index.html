Qt Script for Applications

Qt Script for Applications is a cross-platform toolkit for making C++ applications scriptable using an interpreted scripting language, Qt Script.

You can have a look at Articles about Qt Script for Applications, if you are interested in use cases of QSA or want to submit your own QSA related articles to this forum.
